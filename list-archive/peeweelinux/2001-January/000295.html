Hello,
I was having a little trouble with the "read-only root fs with multiple
ramdisks" I was getting an out of space condition on the / filesystem when
pwl_target_load did its copy of initrd. I found two potential problems with
the root fs space calculation:

1) /home free space as specified is being subtracted from the root fs size
calulation
2) The size of initrd was not added to the root fs size calculation

When I corrected the above, things seemed to work a bit better, thought I
would let you know.

Thanks,
