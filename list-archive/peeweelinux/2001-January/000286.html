About two weeks ago I had a somewhat similiar problem.

There are directions for creating an EXT2FS on the DOC that are
available from M-Systems.  It really is quite easy.   I think it
is a PDF file which I can dig up.

I am using the M-Systems patched lilo for my setup.  I boot off
the DOC and am able to then pick a configuration that has the root
on the DOC or the IDE disk I have.

I am able to do a lilo operation on the DOC even when I am booted
off the disk.   It is just a matter of specifying the proper root
for lilo.

The hardware is a GCT-Allwell system.

I did not use all of the PeeWeeLinux facilities to do this.  I do
have a working system on a 8 MB DOC.  Now all I have to do is 
get the system set up as a router.

into a RAM disk that is loaded at boot time.
