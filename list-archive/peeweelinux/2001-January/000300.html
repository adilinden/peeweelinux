Hi,

A new PeeWeeLinux beta release is now available! Version 0.60.1 includes
the new kernel build feature in pwlconfig. This will allow for a custom
kernel package to be build right from within the pwlconfig environment.
The updated pwlconfig files can be found at http://peeweelinux.org/pub/Projects/Embedded/peeweelinux-0.60.1-cfg.tar.gz 

Important: 0.60.1 is a beta release! The kernel distributed with the
0.60.x release is a must. If an earlier release is updated the kernel
build will fail. In this case the RPMS/kernel-source-2.2.17-* file should
be replaced with the kernel sources found here: http://peeweelinux.org/pub/Projects/Embedded/Embedded_Build/RPMS/kernel-source-2.2.17-11.i386.rpm
