Hm... for me it seems your kernel has no module for ide-disk so it can't
find any root partition. The one solution is to create initial ramdisk with
ide-disk kernel module. On Linuxes like RH 7 you can make it after kernel
compiling with command mkinitrd /boot/initrd-2.2.19.img 2.2.19 and add
statement in lilo.conf under stanza for kernel image. That should be
something like initrd=/boot/initrd-2.2.19.img. The other option is to create
monolithic kernel with all neccessary drivers compiled in...
