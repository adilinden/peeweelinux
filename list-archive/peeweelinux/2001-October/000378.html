run peewee,until appear the interface of "select a
config",then you select one config,then you will find
"make menuconfig",now you open other terminal,delete
the /Embedded_Build/kernel/usr/src/linux,under the
directory of src,tar -zxvf linux-2.4.8.tar.gz(you
should have linux-2.4.8.tar.gz),ok,you back to the old
terminal--peewee interface,now you perform "make
menuconfig",you will find that you are configing the
kernel 2.4.8.I config kernel 2.2.5 with this way is
ok.
