Unfortunately thes are the only docs available at this 
time. There are a number of changes happening to the 
configuration tool when it comes to loading a flash 
system onto the target device. I.e: ramdisks from FAT 
and ext2 will be supported as well as read-only root 
filesystems on the target device.

Some of the new stuff will hopefully be available next 
week some time.

For now I would suggest experimenting with pwlconfig to 
get things going.
