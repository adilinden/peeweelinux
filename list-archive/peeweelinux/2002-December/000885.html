I don't have my system running right now to verify this, but I think
that I see this message a lot when I generate a system.  That file
assists the kernel in finding the library file when a program 
is run.  I don't think it is necessary, it just takes more
time to load a program.  And all the statically linked 
programs don't use libraries at all.

In other words, give it a try and see if it works anyway.
