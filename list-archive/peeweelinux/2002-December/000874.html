Well, my bios does not recognize, correctly, the geometry of my compact
flash card (hitachi)

I need to pass the correct values to the kernel when run lilo.
the configuration file:
--
disk=/dev/hd?
bios=0x80
sectors=32
heads=6
cylinders=306
--
This values are common to most 30MB compact flash.
Just update the values of  CHS to the compact flash
