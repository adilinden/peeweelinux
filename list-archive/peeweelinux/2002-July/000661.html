Ok, success!  I just needed to at the pty entries in the "Configure Project
File System" menu and it works.

Just to recap, required steps:

1) Created the /etc/X11/XF86Config file with FontPath set to the following:
    FontPath "/usr/X11R6/lib/X11/fonts/misc/"
    FontPath "/usr/X11R6/lib/X11/fonts/75dpi/"
    FontPath "/usr/X11R6/lib/X11/fonts/100dpi/"

2) Link XF86_SVGA -> X

3) Added libm, libdl, libncurses libraries

4) Added necessary ptys in /dev

Ed
