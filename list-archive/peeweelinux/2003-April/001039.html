Sure: for starters you need to go into the font directory under /etc/X11 
and edit the config file and remove the listing for 75dpi fonts (you can 
also check that the other fonts listed correspond with real files). Then 
you need to start up xfs (that's the font server). Then you need to link X 
in your /usr/X11R6/bin directory to one of the font servers. Finally, you 
need to configure XF86Config under /usr/X11R6/lib for your graphics 
display and mouse.
