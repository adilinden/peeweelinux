Hi,

The kernel ill mount the root filesystem read-only. Init will run the
system startup scripts which in turn will remount the root filesystem
read-write (if this is the intend).

The "VFS: Mounted root (ext2 filesystem) readonly." line indicates that a
rootfs was indeed found. The question would be, does it contain
"/sbin/init"?

Adi
